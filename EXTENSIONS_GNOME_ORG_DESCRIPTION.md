## Full Description (for extensions.gnome.org)

### Overview
ClipMaster is a powerful, modern clipboard manager for GNOME Shell that helps you manage your clipboard history efficiently. With support for text, images, files, and more, ClipMaster keeps track of everything you copy and makes it easy to access and organize your clipboard items.

### Key Features

#### üìã Clipboard Management
- **Unlimited History**: Keep track of everything you copy with configurable history size (up to 5000 items)
- **Multi-Format Support**: Full support for text, HTML, images, files, URLs, and colors
- **Smart Search**: Real-time search through your entire clipboard history
- **Quick Paste**: Press 1-9 to instantly paste recent items without opening the popup
- **Image Support**: Capture and manage images from clipboard (Wayland & X11)
- **File Tracking**: Track file copies and manage file clipboard entries
- **Primary Selection**: Optionally track PRIMARY selection (middle mouse button paste)

### Installation
1. Download the extension zip file
2. Open GNOME Extensions app
3. Click the gear icon ‚Üí Install from file
4. Select the downloaded zip file
5. Enable the extension

### Requirements
- GNOME Shell 45 or later
- wl-clipboard (for Wayland) or xclip (for X11) for image support

### Usage
- Press `Super+V` (default) to open the clipboard popup
- Click the ClipMaster icon in the top panel
- Use keyboard shortcuts for quick access
- Right-click items for context menu
- Configure everything in the settings panel

### Configuration
Access settings through:
- GNOME Extensions app ‚Üí ClipMaster ‚Üí Settings
- Command: `gnome-extensions prefs clipmaster@gnome.extension`
